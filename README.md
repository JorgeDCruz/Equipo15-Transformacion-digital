## Propuesta
<u>Filtrado de datos:</u> en caso de que se nos entreguen datos en crudo podríamos hacer un filtrado con diferentes herramientas como excel y pytho.

<u>Visualización:</u> se podría utilizar tableau para crear un tablero donde se muestre el análisis de datos

![Tableau](https://www.tableau.com/sites/default/files/2023-07/broadest-deepest-Thumb%402x-cropped.png)

De igual forma se podrían agregar plug-ins a su plataforma de Odoo para hacer un análisis más profundo de los datos que tienen disponibles.

<br>

| Ventajas    | Desventajas |
|----------|----------|
| Facilidad al conectar los plug-ins de Odoo| Tal vez los demás equipos tengan ideas similares|
| Facilidad de manejar los datos | Complicar la parte técnica y que el socio no la pueda implementar (de acuerdo a su infraestructura) |
| Bajo costo de algunas features | Incorporar otro sistema más a los que ya usan cuando una de las herramientas que tienen implementadas podrían hacerlo |
| Aprovechamiento de los datos para obtener una mayor retención del mercado | Aumento en los costos en la parte informática debido a las plataformas |
